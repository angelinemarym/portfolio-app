# Portfolio App

This project was bootstrapped with [Vite](https://vitejs.dev/).

## Design
Based on the Stitch design "Portfolio Home Page - Liquid Glass Variant".
- **Theme**: Dark Mode
- **Font**: Space Grotesk
- **Primary Color**: #137fec

## Setup
1. `npm install`
2. `npm run dev`

## Structure
- `src/App.jsx`: Main layout and sections.
- `src/data.js`: Content for projects, experience, and awards.
- `src/index.css`: Global styles and design tokens.
